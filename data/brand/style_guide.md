# Visual Style Guide

## Color Palette

### Primary Colors
- **Primary**: #6C63FF (Electric Purple)
- **Secondary**: #00D2FF (Cyan)
- **Accent**: #FF6B6B (Coral)

### Dark Mode (Default)
- **Background**: #0F0F1A
- **Surface**: #1A1A2E
- **Card**: #16213E
- **Text Primary**: #FFFFFF
- **Text Secondary**: #B0B0C0

### Light Mode (Alternative)
- **Background**: #F8F9FA
- **Surface**: #FFFFFF
- **Text Primary**: #1A1A2E
- **Text Secondary**: #6B7280

## Typography

### Fonts
- **Headings**: Inter Bold / Space Grotesk Bold
- **Body**: Inter Regular
- **Code/Monospace**: JetBrains Mono
- **Accent**: Outfit (for callouts)

### Size Scale
- **H1 (Cover Title)**: 48-64px, bold
- **H2 (Slide Title)**: 32-40px, bold
- **Body**: 18-22px, regular
- **Caption/Small**: 14-16px

## Visual Style

### Carousel Aesthetic
- Clean, minimal layouts
- Bold typography over subtle backgrounds
- Generous whitespace (breathing room)
- One key visual element per slide (not cluttered)
- Subtle gradients or glow effects (not flat)

### Image Style
- Abstract tech/AI visuals
- Gradient overlays
- Geometric patterns
- Dark mode preferred
- High contrast text overlays

### Icon/Emoji Usage
- Use sparingly and purposefully
- Max 2-3 per caption
- Prefer meaningful emojis (🔍 📊 🧠 ⚡ 🚀) over decorative ones
